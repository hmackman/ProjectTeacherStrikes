U.S. Public School Funding 2000-2015

Overview and Analysis

This visualization represents the state funding per student enrollment per U.S. State for the years of 2000, 2005, 2010, 2015. You can interact with each state to obtain the exact funding per enrollment figure for that state. By clicking on the layered icon on the top right of the page you can change the map layout and toggle between each years data.
